# 第一步升级Android Studio
> 现在，AGP 和 Android Studio 采用基于时间的兼容性政策。 每个 Android Studio 版本都将支持过去 3 年内发布的 AGP 版本。较新的 Android Studio 版本将不再支持 3 年前的 AGP 版本 。如果项目的 AGP 版本不兼容，Android Studio 将要求您更新 AGP。

``` 
AS版本	    所需的     AGP 版本
Quail 1 | 2026.1.1 | 7.1-9.2
Panda 4 | 2025.3.4 | 7.1-9.2
Panda 3 | 2025.3.3 | 7.0-9.1
Panda 2 | 2025.3.2 | 7.0-9.1
Panda 1 | 2025.3.1 | 7.0-9.0
```
